# cocoromusic-by-monProject
あなたの心に最高のボカロ曲を。Android専用ボカロ曲紹介アプリ

気分や質問への回答から、今の気持ちに合うボカロ曲をAIが3曲提案するAndroidアプリです。

## 機能
- 気分入力・アンケートによるAI選曲
- AIが3曲を提案
- YouTube公式音源へのリンク（著作権に配慮）
- いいね機能（※学習は未実装）

## 配布について
- Google Play ストア未配布
- APKを直接インストールする形式です
- 自己責任でインストールしてください

## 対応環境
- Android9以上（9と16で動作確認済み）
