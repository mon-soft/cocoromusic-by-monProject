# cocoromusic-by-monProject
あなたの心に最高のボカロ曲を。GeminiAPIを利用したAndroid専用ボカロ曲紹介アプリ

気分や質問への回答から、今の気持ちに合うボカロ曲をAIが3曲提案するAndroidアプリです。

## 機能
- **気分入力・アンケートによるAI選曲**
  自由に入力して選曲してもらうことも、アンケートに答えてそれに沿ったことを自由入力して相談して高精度な選曲をしてもらうことも可能
- **AIが3曲を提案**
  マッチ度を上部に表示。サムネイルとYoutubeリンクを提供
- **YouTube公式音源へのリンク（著作権に配慮）**
  アプリケーション内データには音声ファイルやサムネイル画像等は一切含んでいません。全て正規の方法で外部参照をしています。公開情報をまとめたデータベースファイルとコードのみです。
- **いいね機能（※学習は未実装）**
  いいねして後で簡単に再生できるようにしよう。ここでのいいねはアプリケーション内でのみ反映されます。Youtubeなどには一切干渉しません。またローカル保存なので開発者は閲覧することはできません。

## 対応環境
- **Android9以上（9と16で動作確認済み）**
  Android8やそれ以前では動作検証を行なっていません。古すぎると動作しない可能性があります。
- **Android専用**
  書いてありますがiOSやWindows等Android以外では一切動作しません。エミュレータ等での動作は想定されていません。
- **インターネット接続必須**
  GeminiAPIの利用やサムネイル取得に利用します。AI関連機能、サムネイル取得、楽曲視聴(Youtubeを利用)にはインターネット接続が必要です。

## 留意事項
- **本アプリの選曲は、あらかじめ収録された楽曲データベースの中から行われます。
各楽曲には、開発者の主観に基づいて調整されたパラメータが設定されており、
ユーザーの入力内容と照合することで、近いと判断された楽曲を提示します。
AIは新しい楽曲を生成・探索するものではなく、
既存データのマッチングを補助する役割として利用されています。また感じ方には個人差があります。**

- **収録楽曲はデータベースで管理されており追加する場合データベース更新が必要になります。これはユーザー側では行うことができません。開発者がアプリの更新をリリースしてユーザーが更新することで実現します。**

- **本アプリは精神的な治療や回復を目的としたものではありません。
AIは利用者の入力内容から、感情や気分に近いと考えられる楽曲を提示するのみです。
音楽の受け取り方や感じ方は人それぞれであり、効果を保証するものではありません。**

- **GeminiAPIは開発者のアカウントの無料枠が利用されています。そのため利用上限というものが設定されており一時的な利用量の増加などで上限に達した場合APIの利用制限がされ、AI機能が利用できなくなる可能性があります。その場合利用制限解除を待つしか選択肢はありません。それによって生じたいかなる損害について開発者は責任を負いません。**
