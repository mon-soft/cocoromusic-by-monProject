# cocoromusic-by-monProject
あなたの心に最高のボカロ曲を。GeminiAPIを利用したAndroid専用ボカロ曲紹介アプリ

気分や質問への回答から、今の気持ちに合うボカロ曲をAIが3曲提案するAndroidアプリです。

## 機能
- 気分入力・アンケートによるAI選曲
  自由に入力して選曲してもらうことも、アンケートに答えてそれに沿ったことを自由入力して相談して高精度な選曲をしてもらうことも可能
- AIが3曲を提案
  マッチ度を上部に表示。サムネイルとYoutubeリンクを提供
- YouTube公式音源へのリンク（著作権に配慮）
  アプリケーション内データには音声ファイルやサムネイル画像等は一切含んでいません。全て正規の方法で外部参照をしています。公開情報をまとめたデータベースファイルとコードのみです。
- いいね機能（※学習は未実装）
  いいねして後で簡単に再生できるようにしよう。ここでのいいねはアプリケーション内でのみ反映されます。Youtubeなどには一切干渉しません。またローカル保存なので開発者は閲覧することはできません。

## 配布について
- Google Play ストア未配布
  ここのページから入手できます。
- APKを直接インストールする形式です
  警告が出る可能性があります。しかし、Android実機での動作確認やPlayプロテクトのスキャンにも合格し、セキュリティ的には問題ありません。署名もしてあります。
- 自己責任でインストールしてください
  アプリケーションに想定外のエラーなどが発生する可能性があります。開発者は修正義務を負いませんし、その問題によってなんらかの損害が生じたとしても開発者は責任を負いません。

## 対応環境
- Android9以上（9と16で動作確認済み）
  Android8やそれ以前では動作検証を行なっていません。古すぎると動作しない可能性があります。
- Android専用
  書いてありますがiOSやWindows等Android以外では一切動作しません。エミュレータ等での動作は想定されていません。
- インターネット接続必須
  GeminiAPIの利用やサムネイル取得にはインターネット接続が必要です。
